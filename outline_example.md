# 细纲格式示例

一个文件可包含多章，每章以 `##` 开头。

## 字段说明

| 字段 | 必须？ | 说明 |
|------|--------|------|
| `## 第N章：标题` | **必须** | 不填无法定位章节，会报错 |
| `**出场人物：**` | 建议填 | 不填则无角色查询、无认知沙箱、无断言核查 |
| `**核心事件：**` | 建议填 | 不填则场景规划缺少关键信息 |
| `**开头：**` | 可选 | 不填为空 |
| `**中间：**` | 可选 | 不填为空 |
| `**结尾：**` | 可选 | 不填为空 |
| `**伏笔：**` | 可选 | 不填为空 |
| `**禁止事项：**` | 可选 | 不填则由Sonnet自行推断forbidden_facts |
| `**字数：**` | 可选 | 不填默认2500字 |

最低限度细纲只需：

```markdown
## 第58章：章节标题

**出场人物：** 角色A、角色B

**核心事件：**
1. 事件一
2. 事件二
```

## 第56章：银叶城竞技场

**出场人物：** 林远、艾伦、莉娅、诺娃

**核心事件：**
1. 铁荆棘在银叶城休整期间得知竞技赛即将开幕
2. 银叶城竞技场报名（团队赛+个人赛）
3. 林远感应到城外废墟方向有微弱的牌共鸣

**开头：** 银叶城竞技场张贴海报——一年一度的银叶城冒险者竞技赛即将开幕，奖金丰厚。莉娅兴奋得差点把海报撕下来。

**中间：** 竞技赛分个人赛和团队赛。铁荆棘报名团队赛+各自报名个人赛。报名处人山人海，林远注意到一个银发青年独自报名D级个人赛。

**结尾：** 回旅馆路上，林远感应到城外东北方向有微弱的库洛牌共鸣。他记下方向，打算找机会去看看。

**伏笔：** 林远感应到城外库洛牌共鸣（剑牌线索）；银发青年身份悬念

**禁止事项：**
1. 不要揭示银发青年的真实身份
2. 不要让林远展示库洛牌的真正能力

**字数：** 2500-3000字

## 第57章：个人赛初赛

**出场人物：** 林远、莉娅、雷欧·阿什顿

**核心事件：**
1. D级个人赛初赛，林远轻松过关
2. 雷欧·阿什顿登场，战斗风格诡异
3. 林远隐约感到雷欧身上有库洛牌的共鸣

**开头：** 竞技场第一天，D级个人赛初赛。三十二人参赛，淘汰赛制，前八名进入正赛。

**中间：** 林远用影牌微弱效果加格斗技巧轻松获胜。休息时注意到雷欧的诡异战斗方式——每次都提前半秒反应，纯靠预判碾压对手。

**结尾：** 雷欧赢下比赛后目光精准落在林远身上，两人对视一秒。林远确认：这个人身上的气息和库洛牌一模一样。

**伏笔：** 雷欧身上的库洛牌气息来源

**禁止事项：**
1. 不要明确雷欧拥有哪张库洛牌
2. 林远不能在初赛中使用库洛牌能力

**字数：** 2500-3000字
